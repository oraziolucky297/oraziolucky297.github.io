💡 About This Project

This tool helps businesses understand which customers might leave. It uses a machine learning model to look at customer data. By finding patterns, the software predicts who needs extra attention. You can use these insights to improve retention and keep your business healthy.

🚀 Getting Started

You do not need programming skills to use this tool. Follow these steps to run the application on your Windows computer.

1. Download the Application

Visit the release page to get the latest file.

Click here to visit the download page. Look for the file ending in .exe and save it to your computer.

2. System Requirements

Your computer must meet these basic needs:

  • Windows 10 or Windows 11.
  • At least 4 gigabytes of memory.
  • An active internet connection for the first run.
  • Administrator rights to install the necessary support files.

3. Running the Tool

Once you finish the download, find the file in your downloads folder. Double-click the file to start the application. A window will open in your internet browser. This window provides the controls for the prediction software. You can upload your data file and view the results directly on the screen.

⚙️ How It Works

The system uses a logistic regression model. This means the computer learns from previous customer history. It balances the data to ensure accuracy for both loyal customers and those at risk. The Streamlit interface displays charts and lists to make the data easy to read.

🛠️ Troubleshooting

If the application does not open, check these items:

  • Check that your computer has enough free space.
  • Ensure your antivirus software is not blocking the file.
  • Restart your computer if the browser window stays blank.
